# Agriculture Produce Price Forecasting AI Web App

An interactive machine learning forecasting web interface powered by Streamlit and Scikit-Learn that auto-downloads time-series agricultural produce price trends directly using `kagglehub`.

## Deployment to Streamlit Cloud

1. Push all project files to a public **GitHub repository**.
2. Sign in to your account dashboard over at [share.streamlit.io](https://share.streamlit.io/).
3. Click on the **"New app"** button.
4. Select your newly created repository, set the branch to `main`, and type `streamlit_app.py` inside the **Main file path** configuration box input.
5. Click **Deploy!** The cloud architecture automatically builds your app environment utilizing the dependencies declared inside `requirements.txt`.
